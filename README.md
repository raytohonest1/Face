# KIWOOM 비대면 본인인증 Demo

Figma 디자인을 반영한 모바일 웹 본인인증 데모입니다. 신분증 촬영, 얼굴 촬영 안내, 좌우 고개 돌림 게이지, 더미 성공 화면으로 이어집니다.

## 실행

브라우저의 카메라 API는 보안 컨텍스트에서만 동작하므로 `file://` 대신 `localhost`로 실행하세요.

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

브라우저에서 `http://127.0.0.1:4173`로 접속합니다.

휴대폰에서 PC 주소로 접속하는 경우 일반 HTTP에서는 카메라 권한이 차단될 수 있습니다. 실제 모바일 테스트는 HTTPS 터널, HTTPS 배포, 또는 기기 내 localhost 환경을 사용하세요.

## 기능

- Figma 기반 모바일 인증 플로우
- 신분증 촬영 화면과 가이드 프레임
- 신분증 촬영 카메라
  - 모바일: 후면 카메라 우선 사용
  - PC 웹: 기본 0번 카메라 사용
- 신분증 인식 대체 로직: 얼굴이 검출된 상태에서 촬영하면 다음 단계로 진행
- 전면 카메라 얼굴 촬영
- 얼굴 yaw 각도 기반 진행
  - 정면 확인
  - 오른쪽 고개 돌림
  - 정면 복귀
  - 왼쪽 고개 돌림
  - 정면 복귀
- 고개를 돌린 각도에 맞춰 원형 게이지 채움/감소
  - 좌회전은 좌측 도넛 게이지 채움
  - 우회전은 우측 도넛 게이지 채움
- 얼굴 비교 없이 `인증에 성공했습니다.` 더미 성공 화면 표시

## 파일

- `index.html`: 앱 진입점
- `styles.css`: Figma 스타일의 모바일 화면, 촬영 프레임, 원형 게이지 스타일
- `app.js`: 화면 상태 전환, 카메라 제어, MediaPipe 얼굴 검출, yaw 기반 게이지 로직

## 외부 모델

MediaPipe WASM과 Face Landmarker 모델은 CDN에서 불러옵니다. 폐쇄망이나 오프라인 환경에서는 `app.js`의 `WASM_ROOT`, `MODEL_URL`을 자체 호스팅 경로로 바꾸세요.
